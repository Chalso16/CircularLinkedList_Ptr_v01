Primer ejercicio de una Lista Circular Enlazda (TAD). Lo implementamos mediante el uso de punteros que apuntan de objeto a objeto, como en una pila, cola o lista. 
Usamos un puntero auxiliar llamado Tail donde este apunta al ultimo elemento y tail->next apunta al primer elemento, de esta manera da igual el tamaño de la lista que el añadir siempre sera de O(1)
